# agents.md

## Objective
Design and develop a professional website focused on Agentic AI frameworks, robotics, UAV systems, ROS 2, and autonomous systems research.

## Agent Roles

### 1. Content Agent
Responsibilities:
- Write technical blog posts
- Prepare educational tutorials
- Generate roadmap descriptions
- Maintain clear technical documentation

### 2. UI/UX Agent
Responsibilities:
- Design modern responsive layouts
- Create dashboard-like research sections
- Optimize readability for technical audiences
- Suggest color palettes and typography

### 3. Research Agent
Responsibilities:
- Collect references and resources
- Track latest developments in robotics and AI
- Organize learning paths from beginner to advanced

### 4. Development Agent
Responsibilities:
- Implement frontend/backend
- Connect CMS or markdown blog system
- Optimize deployment and performance

## Website Tone
Professional, technical, educational, research-oriented.
